# Brief: El Robot

## Objetivo
Mostrar el error invisible que hace que los mensajes de WhatsApp suenen como robot.

## Audiencia
Emprendedores y vendedores que usan WhatsApp para vender.

## Estructura narrativa
1. **Hook**: El error invisible que hace que tu WhatsApp suene como robot
2. **Problema**: Respuestas correctas que no conectan
3. **Ejemplo malo**: Mensaje frio y automatico
4. **Correccion**: Metodo JAPI — preguntar en lugar de informar
5. **Cierre**: No mas, con mas intencion
6. **CTA**: Guarda este reel si vendes por WhatsApp

## Paleta
- Fondo: #F5F5F7
- Accent: #CEF13B
- Texto: #111111

## Duracion
15 segundos (450 frames @ 30fps)
